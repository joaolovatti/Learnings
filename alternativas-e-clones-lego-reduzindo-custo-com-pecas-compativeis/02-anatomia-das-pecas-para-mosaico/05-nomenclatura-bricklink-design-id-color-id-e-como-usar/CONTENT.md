# Nomenclatura BrickLink: Design ID, Color ID e Como Usar

![capa](cover.png)

## Sobre este subcapítulo

Todo o vocabulário desenvolvido até aqui — tipos de peça, medidas, baseplates — precisa de um sistema de identificação preciso para se transformar em um pedido real. No mercado de peças avulsas, essa identificação passa pelo vocabulário do BrickLink: Design ID (o número da forma), Color ID (o número da cor) e Element ID (a combinação dos dois). Sem esse sistema, "1×1 tile amarelo" é uma descrição ambígua que pode resultar na peça errada; com os IDs corretos, o pedido é inequívoco tanto no BrickLink quanto no Gobricks e em qualquer outro fornecedor que use esse padrão.

Este subcapítulo fecha o capítulo porque é a tradução final: pega tudo que o leitor aprendeu sobre tipos de peça, medidas e baseplates e mostra como expressar esse conhecimento no formato que fornecedores entendem. Ao terminar aqui, o leitor está pronto para o próximo capítulo, que apresenta as marcas compatíveis e pressupõe que o leitor já sabe o que está pedindo.

## Estrutura

Os grandes blocos são: (1) Design ID — o identificador da forma da peça, independente de cor; como encontrá-lo no BrickLink e no Gobricks; exemplos concretos para as peças do capítulo (1×1 plate, tile, round plate); (2) Color ID — o identificador numérico de cada cor no sistema BrickLink; como cores LEGO têm nomes oficiais e IDs que diferem dos nomes coloquiais; (3) Element ID — a combinação Design ID + Color ID que identifica uma peça numa cor específica, o que aparece em notas fiscais LEGO e no sistema de pedido do BrickLink; (4) como pesquisar no BrickLink e no Gobricks a partir desses IDs — fluxo prático: dado um tipo e uma cor, encontrar a ficha do produto, verificar o estoque disponível e confirmar que é a peça certa antes de fechar o pedido.

## Objetivo

Ao terminar este subcapítulo, o leitor conseguirá montar uma lista de material com IDs precisos (Design ID + Color ID) para qualquer mosaico, buscar essas peças em BrickLink e Gobricks sem ambiguidade e interpretar fichas de produto de fornecedores que usam a nomenclatura padrão do mercado. Com isso, o capítulo "Anatomia das Peças para Mosaico" estará completo — o leitor tem o vocabulário técnico necessário para operar no ecossistema de marcas compatíveis apresentado no próximo capítulo.
