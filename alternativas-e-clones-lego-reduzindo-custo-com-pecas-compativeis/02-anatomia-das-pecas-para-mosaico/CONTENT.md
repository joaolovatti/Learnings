# Anatomia das Peças para Mosaico

![capa](cover.png)

## Sobre este capítulo

Para comprar peças com precisão — especificar exatamente o que quer, comparar ofertas entre fornecedores e entender por que certas peças custam mais que outras — o leitor precisa falar a língua do sistema LEGO. Este capítulo cobre a nomenclatura e as medidas fundamentais do sistema, com foco nas peças que realmente importam para mosaicos de retrato. Não é um guia enciclopédico de todas as peças existentes; é um mapa cirúrgico do vocabulário necessário para operar neste negócio específico.

A posição no início do livro é intencional: sem saber o que é um 1×1 plate versus um 1×1 tile versus um 1×1 round plate, o leitor não consegue interpretar fichas de produto no BrickLink, entender listas de peças geradas por software ou comparar preços por unidade entre fornecedores.

## Estrutura

Os grandes blocos são: (1) o sistema de medidas LEGO — o que é LDU (LEGO Design Unit), a lógica de studs e a proporção entre brick, plate e tile em altura; (2) as peças essenciais para mosaico — 1×1 plate, 1×1 tile (flat), 1×1 round plate, 1×1 round tile e quando usar cada uma; (3) baseplates — tamanhos disponíveis (16×16, 32×32, 48×48), como se conectam e o papel delas como substrato do mosaico; (4) nomenclatura BrickLink — como as peças são identificadas no mercado (design ID, color ID) e como usar isso para buscar e comprar.

## Objetivo

Ao terminar este capítulo, o leitor conseguirá identificar corretamente qualquer peça relevante para mosaicos pelo nome, entender fichas de produto em lojas como BrickLink e Gobricks, e especificar pedidos sem ambiguidade. Estará pronto para o capítulo seguinte, que apresenta as marcas e pressupõe esse vocabulário.
